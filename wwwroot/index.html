RESTful API for managing quiz questions with full CRUD operations.

API Endpoints

POST /api/quiz
Create a new quiz question
Request Body: { "question": "...", "options": [...], "correctAnswer": 0, "category": "...", "difficulty": "Easy|Medium|Hard" }
GET /api/quiz/{id}
Get a quiz question by ID
Returns the quiz question with the specified MongoDB ObjectId
PUT /api/quiz/{id}
Update an existing quiz question
All fields in request body are optional. Only provided fields will be updated.
DELETE /api/quiz/{id}
Delete a quiz question by ID
Returns 204 No Content on success
